WebTerm:

A Web-based Terminal Emulator and Telnet Client

Version 2.0 (unreleased)

NOTE: The current release version of WebTerm is available on the NACSE home page, at <URL:http://www.nacse.org/web/webterm/>. The version on this page is currently under development, and not supported.

The WebTerm java applet is a terminal emulator which may be embedded within an HTML document or other application which supports java. A low-level screen handler and a Telnet session client are included so that the emulated terminal can be used to establish a remote connection.
